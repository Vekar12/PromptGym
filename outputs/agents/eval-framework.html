LLM Evaluation Framework

Systematic eval harness for agent output quality — v2.1

Eval Dimensions
FaithfulnessReference-free
Does the output contain only claims supported by the provided context? Detects hallucinations and unsupported assertions.
method: nli_entailment(context, output)
Task CompletionRubric-based
Does the output address all sub-tasks in the prompt? Scored 0–5 against a rubric defined per task type.
method: llm_judge(rubric, output)
Format AdherenceStructural
Does the output match the required format (JSON schema, markdown headers, word limits)? Hard pass/fail.
method: schema_validate(output, spec)
Tone ConsistencyEmbedding-based
Is the tone aligned with target persona? Cosine similarity against 20 gold-standard reference outputs.
method: embed_cosine(output, gold_set)
